# Efficient-approximations-of-matrix-multiplication
Codes for the paper "Efficient approximations of matrix multiplication using truncated decompositions" (https://doi.org/10.48550/arXiv.2504.19308) can be found in this repository. The techniques covered in the paper can be used to transform the codes into a more efficient version.

The DFT matrix that was used in the paper can be found at: https://drive.google.com/file/d/1dO-R8atZvRv6Y44AumUbon04aPzLnUVh/view?usp=drive_link
